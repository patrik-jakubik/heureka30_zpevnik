# Zpěvník – veřejné vydání (pouze k nahlédnutí)

Tento repozitář slouží výhradně jako **veřejný zdroj pro zpřístupnění zpěvníku** (PDF) a související jednoduché stránky.

## Důležité
- Repozitář je určen **pouze k nahlédnutí / stažení**.
- **Jakákoliv úprava, přebudování, redistribuce upravené verze nebo jiné zásahy** bez předchozího souhlasu autora jsou **nežádoucí**.
- Nepřijímám **Pull Requesty**, **Issues** ani jiné formy spolupráce přes GitHub.


PDF ve složce repozitáře může být průběžně aktualizováno; odkaz na stránku a QR kód zůstávají stejné.
